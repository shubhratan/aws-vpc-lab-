
## AWS VPC Beginner Lab

This project demonstrates how to design and deploy a secure AWS VPC
with public and private subnets using best practices.

### Architecture
- Custom VPC
- Public & Private Subnets
- Internet Gateway
- NAT Gateway
- Secure Routing

### Skills Demonstrated
- AWS Networking
- Cloud Security Basics
- Subnetting & Routing
- EC2 Access Control

### Outcome
Private EC2 can access the internet securely but is not publicly reachable.
